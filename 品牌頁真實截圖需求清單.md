# 品牌頁真實截圖需求清單

更新日期：2026-07-01

用途：

- 供 9 家品牌頁使用
- 避免用同一組示意圖誤導使用者
- 之後補圖時，會以各品牌真實畫面為準

## 原則

- 每一家品牌頁只放該品牌自己的真實截圖
- 不混用其他品牌畫面
- 不使用共用示意圖假裝成品牌實拍
- 手機截圖與桌面截圖都盡量保留原始 UI
- 如果有會員編號、餘額、個資，先遮掉再給我

## 每一家品牌建議至少提供 3 張

### 1. 品牌大廳列表圖

請截：

- 進入該品牌後的大廳首頁
- 要看得到桌卡列表
- 最好能看出桌型分類或房間列表

用途：

- 放品牌頁開頭
- 對應「大廳動線 / 桌型切換 / 品牌入口閱讀感」

### 2. 單桌桌面圖

請截：

- 真正進入百家樂桌後的完整桌面
- 要看得到：
  - 路單
  - 倒數
  - 下注區
  - 荷官 / 桌面主要資訊

用途：

- 對應「桌面閱讀感 / 畫面觀感 / 桌況理解」

### 3. 手機操作圖

請截：

- 手機版進桌後的畫面
- 要看得到：
  - 下注按鈕
  - 路單
  - 倒數
  - 回桌或切桌入口

用途：

- 對應「手機下注 / 手機閱讀 / 誤觸風險 / 小螢幕體驗」

## 如果能多給，最好再補這 2 張

### 4. 切桌或回大廳流程圖

用途：

- 很適合 `MT / RG / T9 / WG`
- 對應「切桌動線 / 回桌效率 / 多桌觀察」

### 5. 路單清楚度特寫

用途：

- 很適合 `SA / T9 / WG / RG`
- 對應「路單清楚 / 桌況掃讀 / 節奏跟讀」
